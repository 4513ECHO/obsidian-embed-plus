# Embed Plus

[![downloads](https://img.shields.io/badge/dynamic/json?url=https%3A%2F%2Fraw.githubusercontent.com%2Fobsidianmd%2Fobsidian-releases%2Frefs%2Fheads%2Fmaster%2Fcommunity-plugin-stats.json&query=%24.embed-plus.downloads&logo=obsidian&logoColor=%237C3AED&label=Downloads&labelColor=%230e0e0e&color=%237C3AED)](https://community.obsidian.md/plugins/embed-plus)

An Obsidian plugin to embed more external web pages, with external-image syntax and live-preview support.

## Usage

To embed a web page, you can simply use the same Markdown syntax as [external images](https://obsidian.md/help/syntax#External+images):

```markdown
![](https://bsky.app/profile/obsidian.md/post/3lwpb2wjssk2q)
```

Since Obsidian officially supports only [a few kinds of web pages](https://obsidian.md/help/embed-web-pages), this plugin adds support for following web pages:

## Supported web pages

- [Bluesky](https://bsky.app/)
